## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/muneebmughal09/playwright-learning-journey.git
cd playwright-learning-journey
```

### 2. Install Dependencies
```bash
npm install
npx playwright install
```

### 3. Run Examples
```bash
# Run a specific script
node week-01-core-basics/01-setup-and-config/code.js

# Run with debug mode
PWDEBUG=1 node week-01-core-basics/01-setup-and-config/code.js
```
